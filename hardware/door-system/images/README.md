# Door System Images

This directory is for door system specific diagrams, wiring schematics, and photos.

## Planned Images

- **wiring-diagram.png** - Complete wiring diagram showing all connections
- **pinout-diagram.png** - ESP32 pinout with labeled connections
- **rfid-connections.png** - Detailed RFID reader wiring
- **lcd-connections.png** - LCD display wiring details
- **servo-mounting.jpg** - Example servo installation
- **final-installation.jpg** - Complete door system setup
- **component-layout.jpg** - Physical component arrangement

## Image Guidelines

- Use clear, high-resolution images
- Include pin numbers and labels
- Use color-coded wires in diagrams
- Save in PNG format for diagrams, JPG for photos
