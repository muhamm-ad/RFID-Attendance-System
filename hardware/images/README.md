# Architecture Images

This directory is for general system architecture diagrams and images.

## Planned Images

- **system-architecture.png** - Overall system architecture diagram showing both ESP32 systems and backend server
- **network-topology.png** - Network connectivity diagram
- **component-overview.png** - Overview of all hardware components

## Image Guidelines

- Use clear, high-resolution images
- Include labels and annotations
- Use consistent color scheme
- Save in PNG format for diagrams, JPG for photos
