# Registration System Images

This directory is for registration system specific diagrams, wiring schematics, and photos.

## Planned Images

- **wiring-diagram.png** - Complete wiring diagram for registration system
- **rfid-connection.png** - Detailed RFID reader wiring
- **setup-photo.jpg** - Complete registration system setup
- **component-layout.jpg** - Physical component arrangement
- **integration-workflow.png** - Workflow diagram showing badge scan to registration

## Image Guidelines

- Use clear, high-resolution images
- Include pin numbers and labels
- Use color-coded wires in diagrams
- Save in PNG format for diagrams, JPG for photos
